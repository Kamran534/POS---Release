# TradeUnleashed - POS Desktop App

Download page for the TradeUnleashed desktop POS application.

## Download

Visit the [download page](https://kamran534.github.io/POS---Release/) or go directly to [Releases](https://github.com/Kamran534/POS---Release/releases).

**Latest version:** 1.0.0

## System Requirements

- Windows 10 or later (64-bit)
- 4 GB RAM minimum
- 200 MB disk space

## Installation

1. Download `TradeUnleashed-Setup-1.0.0.exe` from the [latest release](https://github.com/Kamran534/POS---Release/releases/latest)
2. Run the installer
3. If Windows SmartScreen appears, click **"More info"** then **"Run anyway"**
4. The app installs and launches automatically
